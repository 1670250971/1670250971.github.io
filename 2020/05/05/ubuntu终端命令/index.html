ubuntu终端命令

1、文件与目录

分区：Linux没有分区概念

Linux以目录的组织，只有根目录/

/bin ：可执行文件

/sbin: 供超级用户使用的可执行文件

s=super

/etc: 配置文件

eg: 源地址： /etc/apt/sources.list

/home: 家目录，给用户存储文件用

-每一个用户都有一个用户名命名的文件夹

-收藏夹中的主目录=用户自己的家目录

-同时可以看收藏夹中也有家目录各子目录的快捷方式

/mnt: 挂载设备用

/boot: 引导程序

操作系统：内核->系统调用->内置的命令行或基本的图形程序->第三方应用->用户

二、Linux终端与命令

1、ls

-l :详细输出文件或目录信息

-h :人性化方式显示文件大小

Bytes<KB< MB <GB <TB <PB<EB <ZB<YB

-a :显示隐藏文件（以.开始）

.： 当前目录

.. : 上一级目录

~： 表示用户自己的家目录

= /home/username

ls -a -h = ls -al

2、 pwd

print working directory

pwd : 显示当前的工作目录

3、cd 切换文件夹

change directory

cd /

cd ~

cd ..

Linux中的命令区分大小写

4、touch

格式：touch 文件名

如果文件不存在则新建一个

如果文件存在则修改最近修改时间

5、mkdir

创建目录make directory

支持同时创建多个文件夹：

-同级目录，用空格间隔 mkdir a b

-层级目录，用-p参数 mkdir -p a/b/

同级目录下，文件与文件夹不能重名

6、rmdir

删除空目录remove directory

终端中删除的内容不进回收站

7、rm

删除文件remove

可以使用通配符：

–*:任意多字符

–?:一个字符

rm -r 非空目录名： 删除非空目录

终端中删除的内容不进回收站！！

8、clear

清屏

9、技巧：

（1)利用上下键

(2)使用Tab: 命令与路径自动补全

(3)区分相对路径与绝对路径：

-绝对路径：从/开始的路径

因为=/home/username，所以开头的也称为绝对路径

-相对路径：从当前目录开始的路径
